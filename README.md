# Codenames iOS Game
A simple SwiftUI implementation of the board game Codenames.
This project was developed for Iteration 1, focusing on the core game mechanics and architecture.

Features (Iteration 1)：

·5×5 random board generation
·Spymaster clue submission
·Operative card selection
·Turn switching
·Assassin handling
·Win/Loss detection
·Game Over screen


Architecture：

The project follows a basic MVVM structure:
·Domain – Game models and logic (GameService)
·Presentation – Views and ViewModels
·SwiftUI – User interface

Game logic is separated from UI.


How to Run：

·Open the project in Xcode
·Select an iPhone simulator
·Press ⌘ + R
